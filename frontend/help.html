Getting Started

+

Welcome to GetItDone! Getting started is simple and takes less than 2 minutes.

Creating Your Account

  1. Login or Create Account: Click the login button and choose to create a new account
  2. Enter Valid Email & Password: Use a real email address as you'll need to verify it
  3. Check Your Inbox: Look for a confirmation email (check spam folder if not found)
  4. Click Confirmation Link: This will automatically sign you in and load your homepage

Using the App

+

Step 1: Enable Browser Notifications

When you first use GetItDone, your browser will ask for permission to send notifications. This is essential for task reminders.

Click "Allow" when this notification appears

Step 2: Enable App Notifications

After allowing browser notifications, configure the app's notification settings:

  1. Click the Settings button in the top right corner
  2. Check the box labeled "Enable Task Reminders"
  3. Optionally add your name for personalized notifications
  4. Click Save to apply your settings

Settings panel - enable notifications here

Step 3: Start Managing Tasks

You're all set! Use the default loaded tasks or add your own. The app handles everything else automatically.

Troubleshooting & FAQ

+

I'm not receiving any task reminders

First, check your app settings and ensure you have ticked "Enable Task Reminders". Additionally, if you installed the app via Brave browser or Firefox, they block notifications by default. Consider installing the app via Chrome for full notification support.

I didn't receive the confirmation email/reset password link

Ensure you entered a valid email address and check your spam folder. The email should arrive within a few minutes.

What if my tasks are recurring?

Don't worry! All tasks are recurring by default unless you explicitly delete them. All task progress resets at midnight, and their status is saved and accessible via the "View Report" button at the top menu. This saves task progress for the past 7 days, including those completed on time, completed late, or missed. The system resets all task progress fresh each midnight because each new day should start on a fresh state.

Why do some tasks appear greyed out and I can't mark them as complete?

The system checks the end time of each task and gives a 15-minute grace period to mark them as complete. If not completed within this time, the task is greyed out and a new "Complete Late" button appears on the task card. If clicked, you must enter a reason why the task is late, upon which it will be marked as complete. The system does this to enforce accountability in a soft way. However, this task will be tracked in the View Report section as having been completed late.

Late task completion - accountability feature

Can I login with the same account across multiple devices?

Absolutely! Cross-device syncing is supported. Changes made on one device will automatically reflect on any logged-in device in real time.

What happens when I'm offline and make changes or add tasks?

The system uses an offline-first architecture - that's one of our key goals. Don't worry, do your work! Any changes you make will be queued and synced to the database when you're back online. Our goal is to handle the heavy lifting for you, so you can handle what really matters.

Can I delete or edit tasks?

Absolutely! Use the edit and delete buttons available on each task card.

I have a suggestion - how can I contact the developer?

Scroll to the end of this FAQ and you'll see a contact section. Send me a message - I reply fast! Alternatively, at the footer of any page in this app, you'll find a "Contact Developer" link.
